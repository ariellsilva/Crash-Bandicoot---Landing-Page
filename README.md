# 🎮 Crash Bandicoot Hub - Landing Page

Esta é uma landing page responsiva e interativa dedicada à franquia **Crash Bandicoot**. O projeto foi desenvolvido como parte do meu portfólio para demonstrar habilidades em **Front-end Development**, com foco em design moderno, interatividade e adaptação para dispositivos móveis.

## 🚀 Funcionalidades

* **Design Responsivo**: Layout totalmente adaptado para telas de até **820px**, garantindo uma boa experiência em tablets e smartphones.
* **Navegação Fixa**: Menu de navegação estilizado com a cor vibrante da franquia e posicionado no topo para facilitar o acesso.
* **Seção de Criadores**: Destaque para o estúdio **Naughty Dog**, com informações sobre a origem da série.
* **Vídeos Integrados**: Galeria de vídeos do YouTube sobre a jogabilidade rápida e divertida da franquia.
* **Formulário de Contato**: Área dedicada para feedback e interação com o usuário, estilizada com a identidade visual do jogo.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica das seções (Sobre, Jogos, Personagens, Criadores e Contato).
* **CSS3**:
* Uso de **Media Queries** para responsividade.
* Estilização com fontes temáticas como **'Bangers'** e **'Luckiest Guy'**.
* Efeitos de **Flexbox** e **Gradients** personalizados.


* **JavaScript**: Implementação de lógica para exibição de modais e interações dinâmicas.

## 📱 Desafios Superados

Durante o desenvolvimento, foquei em resolver problemas complexos de **UI/UX**, como:

1. **Ajuste de Títulos no Mobile**: Correção de sobreposição de elementos e ajustes de `font-size` para evitar cortes em telas pequenas.
2. **Posicionamento Relativo**: Substituição de posicionamentos fixos por layouts flexíveis para garantir que o conteúdo não "sumisse" ou ficasse desalinhado.

## 🎨 Identidade Visual

O projeto utiliza uma paleta de cores inspirada no universo do Crash, com tons de laranja vibrante e azul, além de tipografias que remetem à estética de aventura e ação dos jogos de plataforma.
